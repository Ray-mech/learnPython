演習問題
```python3
#四則演算と条件分岐を用いてaの絶対値を表示するプログラムをつくろう
a = int(input())
if a < 0:
    answer = -a
else:
    answer = a
string = str(a) + "の絶対値は" + str(answer) + "です！"
print(string)
```
